## 摘要
现有的语言驱动具身导航范式在具有高度相似特征的功能性建筑（FBs）中面临挑战，因为它们缺乏有效利用先验空间知识的能力。为了解决这一问题，我们提出了一种先验地图引导的具身导航（PM-Nav），其中环境地图被转换为导航友好的语义先验地图，设计了一种带有标注先验地图的分层思维链提示模板以实现精确的路径规划，并构建了一个多模型协作动作输出机制，以完成导航规划的定位决策和执行控制。使用自制 FB 数据集的全面测试表明，PM-Nav 在模拟和真实世界中相较于 SG-Nav 和 InstructNav 分别获得了 511% 和 1175%、以及 650% 和 400% 的平均提升。这些巨大的提升阐明了将 PM-Nav 作为功能性建筑骨干导航框架的巨大潜力。

## Abstract
Existing language-driven embodied navigation paradigms face challenges in functional buildings (FBs) with highly similar features, as they lack the ability to effectively utilize priori spatial knowledge. To tackle this issue, we propose a Priori-Map Guided Embodied Navigation (PM-Nav), wherein environmental maps are transformed into navigation-friendly semantic priori-maps, a hierarchical chain-of-thought prompt template with an annotation priori-map is designed to enable precise path planning, and a multi-model collaborative action output mechanism is built to accomplish positioning decisions and execution control for navigation planning. Comprehensive tests using a home-made FB dataset show that the PM-Nav obtains average improvements of 511\% and 1175\%, and 650\% and 400\% over the SG-Nav and the InstructNav in simulation and real-world, respectively. These tremendous boosts elucidate the great potential of using the PM-Nav as a backbone navigation framework for FBs.

#### 1. 论文的核心问题与整体含义（研究动机和背景）
现有的具身导航（Embodied Navigation）研究多集中在结构简单的家庭室内环境。但在**功能性建筑（Functional Buildings, FBs）**（如学校、医院、政府大楼）中，导航面临巨大挑战：
*   **特征高度相似**：走廊、房间门等视觉特征极其雷同，缺乏个性化地标。
*   **空间逻辑复杂**：开放空间大、结构复杂，传统的视觉语言模型（VLM）难以直接通过图像逻辑推理理解复杂的空间关系。
*   **缺乏先验知识利用**：现有方法多为盲目探索或依赖离散节点，无法像人类一样利用地图（先验知识）进行全局规划。
**核心动机**：模仿人类利用地图和地标在陌生大型建筑中导航的行为，提出一种由先验地图引导的导航框架 PM-Nav。

#### 2. 论文提出的方法论
PM-Nav 框架包含三个核心阶段：
*   **地图解析（Map Parsing）**：
    *   将原始环境地图转化为“导航友好”的**结构化语义先验图**。
    *   定义关键航路点（起点、转弯、分支、终点）并将路径划分为段（Segments），建立房间 ID 与路径段的拓扑关联，降低 VLM 的认知负荷。
*   **VLM 路径规划（VLM Planning）**：
    *   采用**分层思维链（H-CoT）提示模板**，结合带有标注的先验地图输入给 VLM（如 GPT-4o）。
    *   规划过程分为：确定当前位置与目标的空间关系 -> 识别必经航路点 -> 生成第一人称导航动作序列。
*   **多模型协作动作生成（Action Generation）**：
    *   **粗粒度动作**：VLM 根据全景图像识别地标并给出大致方向（如每 30° 一个区间）。
    *   **细粒度动作**：利用 **GroundingDINO** 和 **SAM** 精确标记目标地标，再通过微调的 **PixelNav** 神经网络输出精确的转向角度。
    *   **闭环机制**：形成“目标确定 → 搜索 → 验证 → 目标更新”的循环，确保代理持续向目标推进。

*   **多模型协作的必要性**：仅靠 VLM 的粗粒度输出不足以实现精确导航，结合底层视觉模型（SAM/DINO）的细粒度微调是成功的关键。

*   **场景针对性强**：首次深入探讨了具身导航在“功能性建筑”这一特殊且高难度场景下的应用。
*   **人类启发式设计**：成功将人类“看地图+找地标”的认知模式转化为机器可执行的算法流程。
*   **模块化协作**：巧妙地整合了高层逻辑推理（VLM）与底层感知控制（PixelNav/SAM），实现了端到端的导航闭环。
