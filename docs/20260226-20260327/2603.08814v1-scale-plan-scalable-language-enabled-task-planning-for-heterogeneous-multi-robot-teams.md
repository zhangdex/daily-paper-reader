## 摘要
异构多机器人系统的长时程任务规划对于在现实环境中部署协作团队至关重要；然而，由于感知信息量巨大，且其中大部分与任务目标无关并加重了规划负担，这仍然具有挑战性。传统的符号规划器依赖于人工构建的问题规范，限制了可扩展性和适应性，而近期基于大语言模型（LLM）的方法在物体丰富的场景中往往面临幻觉和弱具身落地（grounding）问题——即生成的计划与实际环境物体及约束之间的对齐较差。我们提出了 Scale-Plan，这是一个可扩展的 LLM 辅助框架，能够根据自然语言指令生成紧凑且与任务相关的问题表示。给定 PDDL 领域规范，Scale-Plan 构建一个捕捉领域结构的动作图，并利用浅层 LLM 推理来引导结构化图搜索，从而识别相关动作和物体的最小子集。通过在规划前过滤无关信息，Scale-Plan 实现了高效的任务分解、分配和长时程计划生成。我们在复杂的多智能体任务上评估了我们的方法，并介绍了 MAT2-THOR，这是一个基于 AI2-THOR 构建的经过清洗的基准测试集，用于对多机器人规划系统进行可靠评估。Scale-Plan 在所有指标上均优于纯 LLM 和混合 LLM-PDDL 基线，提升了可扩展性和可靠性。

## Abstract
Long-horizon task planning for heterogeneous multi-robot systems is essential for deploying collaborative teams in real-world environments; yet, it remains challenging due to the large volume of perceptual information, much of which is irrelevant to task objectives and burdens planning. Traditional symbolic planners rely on manually constructed problem specifications, limiting scalability and adaptability, while recent large language model (LLM)-based approaches often suffer from hallucinations and weak grounding-i.e., poor alignment between generated plans and actual environmental objects and constraints-in object-rich settings. We present Scale-Plan, a scalable LLM-assisted framework that generates compact, task-relevant problem representations from natural language instructions. Given a PDDL domain specification, Scale-Plan constructs an action graph capturing domain structure and uses shallow LLM reasoning to guide a structured graph search that identifies a minimal subset of relevant actions and objects. By filtering irrelevant information prior to planning, Scale-Plan enables efficient decomposition, allocation, and long-horizon plan generation. We evaluate our approach on complex multi-agent tasks and introduce MAT2-THOR, a cleaned benchmark built on AI2-THOR for reliable evaluation of multi-robot planning systems. Scale-Plan outperforms pure LLM and hybrid LLM-PDDL baselines across all metrics, improving scalability and reliability.

在现实世界的复杂环境中，异构多机器人系统的长程任务规划面临两大挑战：
*   **感知信息冗余：** 环境中存在大量与当前任务无关的物体和状态信息，这会显著增加规划器的搜索空间，导致效率低下。
*   **现有方法的局限：** 传统的符号规划器（如 PDDL）依赖人工编写问题文件，缺乏灵活性；而纯大语言模型（LLM）在处理多物体场景时容易产生幻觉，且生成的计划往往与物理环境约束脱节（即“弱具身落地”）。
**研究动机：** 开发一种能够自动过滤无关环境信息、生成紧凑任务表示，并能高效协调异构机器人团队的规划框架。

#### 2. 方法论：核心思想与关键技术
Scale-Plan 的核心思想是**“先过滤、后规划”**，通过结合符号逻辑的严谨性和 LLM 的推理能力实现可扩展性。
*   **动作图构建（Action Graph Construction）：** 离线从 PDDL 领域规范中提取动作模式。节点为参数化动作，边表示逻辑依赖。
    *   *严格规则：* 若动作 A 的效果完全满足动作 B 的前提，则连边。
    *   *松散规则：* 若有部分前提重叠且无严格连接，则连边以保持连通性。
*   **任务相关过滤（Environment Filtering）：** 在运行时，利用浅层 LLM 推理识别任务目标相关的候选动作，随后在动作图上进行**逆向深度优先搜索（DFS）**，提取实现目标所需的最小动作和物体子集。
*   **结构化规划流水线：**
    1.  **任务分解：** 将高层指令分解为子任务序列。
    2.  **机器人分配：** 根据异构机器人的能力（技能集）分配子任务。
    3.  **计划整合：** 分析子任务间的依赖，确定并行或串行执行顺序。
    4.  **代码化执行：** 将计划转换为 AI2-THOR 模拟器可执行的指令。

#### 5. 实验数量与充分性
*   **实验规模：** 针对 49 个不同难度的任务进行了端到端测试，并进行了详细的**消融实验**（对比了无过滤、联合规划、浅层 LLM 过滤等变体）。

#### 6. 主要结论与发现
*   **性能领先：** Scale-Plan 在所有指标上均优于基线。在复杂任务中，其任务完成率（TCR）比最强基线高出 35%。
*   **过滤的重要性：** 过滤掉无关物体能显著减少 LLM 的幻觉，提高长程规划的可靠性。
*   **效率权衡：** 虽然结构化规划（多次 LLM 调用）比直接生成计划耗时更长，但换取了极高的成功率和可执行性。

#### 7. 优点与亮点
*   **动作图机制：** 巧妙地利用 PDDL 的结构化知识来引导 LLM，解决了纯神经方法难以处理的逻辑依赖问题。
*   **异构支持：** 显式考虑了不同机器人的技能差异，增强了团队协作的灵活性。
*   **数据集贡献：** 发布的 MAT2-THOR 修正了前人工作的错误，为多机器人规划领域提供了更可靠的 Benchmark。

#### 8. 不足与局限
*   **语义推理缺失：** 偶尔会忽略物体的“可供性”（Affordance），例如在放入物体前忘记打开柜门。
*   **操作约束建模不足：** 存在机器人未放下手中物体就尝试抓取新物体的逻辑错误。
*   **定位幻觉：** 尽管有过滤机制，LLM 仍可能在物体具体空间位置的感知上产生幻觉。
*   **实时性挑战：** 多次 LLM 推理导致规划时间（PT）较长，可能不适用于对实时响应要求极高的动态场景。
