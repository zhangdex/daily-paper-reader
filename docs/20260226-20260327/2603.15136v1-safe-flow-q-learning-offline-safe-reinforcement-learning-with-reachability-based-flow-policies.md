## 摘要
离线安全强化学习（RL）旨在严格的安全约束下，从静态数据集中寻求奖励最大化的策略。现有方法通常依赖于软期望成本目标或迭代生成式推理，这对于安全关键型实时控制可能是不够的。我们提出了 Safe Flow Q-Learning (SafeFQL)，它通过结合受 Hamilton-Jacobi 可达性启发的安全价值函数和高效的一步流策略，将 FQL 扩展到安全离线强化学习。SafeFQL 通过自一致性 Bellman 递归学习安全价值，通过行为克隆训练流策略，并将其蒸馏为一步 Actor，以便在部署时无需拒绝采样即可进行奖励最大化的安全动作选择。为了解决学习到的安全边界中存在的有限数据近似误差，我们增加了一个共形预测校准步骤，该步骤调整安全阈值并提供有限样本概率安全覆盖。实验表明，与扩散式安全生成基线相比，SafeFQL 以略高的离线训练成本换取了显著降低的推理延迟，这有利于实时安全关键型部署。在小船导航和 Safety Gymnasium MuJoCo 任务中，SafeFQL 在大幅减少约束违反的同时，达到或超过了先前的离线安全强化学习性能。

## Abstract
Offline safe reinforcement learning (RL) seeks reward-maximizing policies from static datasets under strict safety constraints. Existing methods often rely on soft expected-cost objectives or iterative generative inference, which can be insufficient for safety-critical real-time control. We propose Safe Flow Q-Learning (SafeFQL), which extends FQL to safe offline RL by combining a Hamilton--Jacobi reachability-inspired safety value function with an efficient one-step flow policy. SafeFQL learns the safety value via a self-consistency Bellman recursion, trains a flow policy by behavioral cloning, and distills it into a one-step actor for reward-maximizing safe action selection without rejection sampling at deployment. To account for finite-data approximation error in the learned safety boundary, we add a conformal prediction calibration step that adjusts the safety threshold and provides finite-sample probabilistic safety coverage. Empirically, SafeFQL trades modestly higher offline training cost for substantially lower inference latency than diffusion-style safe generative baselines, which is advantageous for real-time safety-critical deployment. Across boat navigation, and Safety Gymnasium MuJoCo tasks, SafeFQL matches or exceeds prior offline safe RL performance while substantially reducing constraint violations.

#### 1. 核心问题与整体含义
在安全关键型领域（如机器人、自动驾驶），在线强化学习的试错成本极高，因此**离线安全强化学习（Offline Safe RL）**成为研究重点。然而，现有方法面临两大挑战：
*   **安全性与实时性的矛盾**：传统的拉格朗日法仅能保证“期望安全”而非“瞬时硬约束”；而基于扩散模型（Diffusion）的生成式策略虽然表达力强，但推理时需要多次迭代去噪，延迟高，难以满足高频实时控制需求。
*   **离线数据的分布偏移与误差**：离线学习的价值函数在有限数据下存在近似误差，导致预估的安全边界不可靠。

本文提出的 **SafeFQL** 旨在通过结合 **Hamilton-Jacobi (HJ) 可达性理论**与**流匹配（Flow Matching）技术**，构建一个既能保证严格安全边界，又能实现单步高效推理的离线强化学习框架。

#### 2. 方法论
SafeFQL 的核心流程分为四个阶段：
*   **学习奖励与安全评论家（Critics）**：
    *   **奖励 Critic**：采用隐式 Q 学习（IQL），通过分位数回归学习数据集内的最优奖励。
    *   **安全 Critic**：受 HJ 可达性启发，采用**最大备份贝尔曼递归（Max-backup Bellman recursion）**。公式定义为 $y_c = \max(\ell(x), \gamma V_c(x'))$，其中 $\ell(x)$ 是即时安全信号。这确保了只要未来任何时刻可能进入危险区域，当前状态就会被标记为不安全。
*   **流策略蒸馏**：
    *   首先训练一个基于流匹配的行为策略作为“教师”。
    *   将其蒸馏为一个**一步学生 Actor（One-step Actor）**。该 Actor 直接将状态和噪声映射到动作，避免了推理时的迭代去噪。
*   **可行性门控目标（Feasibility-Gated Objective）**：
    *   引入一个二值门控机制：如果当前动作被预测为不安全（$Q_c \ge 0$），Actor 仅优化安全目标以回到安全区；如果预测安全（$Q_c < 0$），则同时优化奖励和行为正则化。这种“优先级排序”避免了奖励与安全梯度冲突导致的训练不稳定。
*   **共形预测（Conformal Prediction）校准**：
    *   利用留出集（Held-out set）进行统计校准，自动调整安全阈值 $\delta$，为部署提供有限样本下的概率安全覆盖保证。
