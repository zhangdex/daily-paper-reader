## 摘要
通过明确且可操作的里程碑来衡量任务进度，对于鲁棒的机器人操作至关重要。这种进度感知能力使模型能够确定当前任务状态，预测可验证的中间状态，并在进度停滞时检测并从故障中恢复。为了实现这一能力，我们提出了 See, Plan, Rewind (SPR)，这是一个进度感知的视觉-语言-动作框架，它能动态地将语言指令转化为一系列空间子目标。SPR 通过一个持续的核心循环运行：观察（Seeing）当前状态和即将到来的里程碑，规划（Planning）通往下一个 2D 路径点的轨迹，以及在失败时通过对照预期序列监测进度来回溯（Rewinding）到可恢复状态。这种闭环方法实现了鲁棒的错误纠正，且无需额外的训练数据或辅助模型。大量实验证明了该框架的有效性、泛化性和鲁棒性：在 LIBERO 基准测试中，SPR 的表现优于 MolmoAct 基准模型 5%。在具有未见指令和初始状态的挑战性 LIBERO-Plus 基准测试中，SPR 以最小的性能下降实现了最先进的鲁棒性，超越了 OpenVLA-OFT 和 UniVLA，展示了卓越的分布外鲁棒性。

## Abstract
Measurement of task progress through explicit, actionable milestones is critical for robust robotic manipulation. This progress awareness enables a model to ground its current task status, anticipate verifiable intermediate states, and detect and recover from failures when progress stalls. To embody this capability, we introduce See, Plan, Rewind (SPR), a progress-aware vision-language-action framework that dynamically grounds language instructions into a sequence of spatial subgoals. SPR operates through a continuous core cycle, Seeing the current state and upcoming milestone, Planning a trajectory towards the next 2D waypoint, and Rewinding to a recoverable state upon failure by monitoring progress against the expected sequence. This closed-loop approach enables robust error correction without requiring additional training data or auxiliary models. Extensive experiments demonstrate the framework's effectiveness, generalization and robustness: SPR outperforms the MolmoAct baseline by 5\% on the LIBERO benchmark. On the challenging LIBERO-Plus benchmark with unseen instructions and initial states, SPR achieves state-of-the-art robustness with the smallest performance drop, surpassing OpenVLA-OFT and UniVLA, demonstrating superior out-of-distribution robustness.

*   **整体含义**：论文提出了 **SPR (See, Plan, Rewind)** 框架，通过将任务分解为带有空间坐标的子目标序列，实现了闭环的进度监控与错误恢复，显著提升了机器人操作的鲁棒性和分布外（OOD）泛化能力。

### 2. 方法论
SPR 框架由三个核心阶段构成的闭环循环组成：
*   **See（观察与感知）**：模型接收图像和指令，预测剩余子任务的数量、每个子任务的语义描述及其在图像中的 **2D 空间坐标**。这为任务进度提供了可量化的基准。
*   **Plan（规划）**：模型不直接规划到最终目标，而是规划通往**下一个子目标**的 2D 轨迹（包含 1-5 个路径点）。这种细粒度规划降低了长程任务的复杂度。
*   **Rewind（回溯恢复）**：
    *   **异常检测**：通过状态记录器监控预测的子任务数和轨迹。如果子任务数增加（倒退）或轨迹长时间不变（停滞），则触发异常。
    *   **恢复策略**：一旦检测到异常，模型切换到“回溯”模式，执行 $N=3$ 步“返回初始位置”的动作，将机械臂撤回到分布内（In-distribution）的安全状态，随后重新尝试任务。
*   **关键技术细节**：
    *   **自动数据标注**：利用 DINOv3 和 SAM 自动提取演示视频中的夹具轨迹和空间坐标；利用 Gemini-3 或 DeepSeek-R1 进行子任务语义分割和描述生成，无需人工标注。
    *   **统一模型**：基于 MolmoAct 架构，将深度感知、进度预测、空间规划和动作生成集成在单个自回归序列中。

*   **实验规模**：涵盖了从仿真到真机的全方位评估。在 LIBERO-Plus 上的大规模测试（数千个变体）极具说服力。

### 6. 主要结论与发现
*   **性能提升**：SPR 在 LIBERO 平均成功率达到 91.8%，比 MolmoAct 提升了 5%，尤其在长程任务（Long）中表现突出。
*   **卓越的鲁棒性**：在 LIBERO-Plus 的 OOD 测试中，SPR 的性能下降幅度最小（平均下降 18.8%，而其他模型如 OpenVLA-OFT 下降 27%），证明了空间子目标对环境变化的抵抗力。
*   **真机表现**：在复杂的“整理桌面”任务中，当物体增加到 3 个以上时，基准模型全部失败，而 SPR 仍能保持一定的成功率。
*   **回溯的价值**：Rewind 机制能有效减少重新执行失败子任务所需的时间，提高任务完成效率。

### 7. 优点
*   **显式空间接地**：将抽象的进度监控转化为具体的 2D 坐标预测，使模型的决策过程具有高度的可解释性。
*   **数据高效性**：无需专门收集失败/恢复数据，仅通过翻转成功演示数据即可训练回溯能力。
*   **闭环纠错**：不依赖外部辅助模型（如 GPT-4V）进行重规划，而是通过内在的进度感知实现自主恢复。

### 8. 不足与局限
*   **精度限制**：受限于离散化的动作 Token，在需要毫米级精度的微小操作（如精准对齐）中仍存在局限。
*   **物理受限**：如果机器人被物理卡死（如夹具被物体卡住），单纯的 Rewind 指令可能无法产生位移，导致恢复失效。
*   **动作-规划不一致**：在极少数情况下，模型预测的 2D 轨迹正确，但生成的底层动作 Token 却未遵循该轨迹（存在幻觉风险）。
