## 摘要
在部分可观测环境（并非所有物体都已知或可见）中进行机器人规划是一个具有挑战性的问题，因为它需要通过部分可观测马尔可夫决策过程在不确定性下进行推理。在执行计算出的规划期间，机器人可能会意外观察到与任务无关的物体，而这些物体通常会被朴素规划器忽略。在这项工作中，我们提出结合两种常识性知识：（1）某些物体更有可能出现在特定位置；（2）相似的物体更有可能共同出现，而不同的物体则不太可能出现在一起。手工构建此类知识非常复杂，因此我们探索利用大语言模型（LLM）强大的常识推理能力。我们的规划与执行框架 CoCo-TAMP 引入了一种分层状态估计，利用 LLM 引导的信息来塑造对任务相关物体的信念，从而能够高效解决长时程任务与运动规划问题。在实验中，与不包含任何一种常识性知识的基准方法相比，CoCo-TAMP 在仿真中的规划和执行时间平均减少了 62.7，在真实世界演示中减少了 72.6。

## Abstract
Robot planning in partially observable environments, where not all objects are known or visible, is a challenging problem, as it requires reasoning under uncertainty through partially observable Markov decision processes. During the execution of a computed plan, a robot may unexpectedly observe task-irrelevant objects, which are typically ignored by naive planners. In this work, we propose incorporating two types of common-sense knowledge: (1) certain objects are more likely to be found in specific locations; and (2) similar objects are likely to be co-located, while dissimilar objects are less likely to be found together. Manually engineering such knowledge is complex, so we explore leveraging the powerful common-sense reasoning capabilities of large language models (LLMs). Our planning and execution framework, CoCo-TAMP, introduces a hierarchical state estimation that uses LLM-guided information to shape the belief over task-relevant objects, enabling efficient solutions to long-horizon task and motion planning problems. In experiments, CoCo-TAMP achieves an average reduction of 62.7 in planning and execution time in simulation, and 72.6 in real-world demonstrations, compared to a baseline that does not incorporate either type of common-sense knowledge.

#### 1. 核心问题与整体含义（研究动机和背景）
在现实世界中，机器人执行长程任务（如“去厨房拿个苹果”）时，往往面临**部分可观测性（Partial Observability）**的问题：物体可能被遮挡，或者其具体位置未知。传统的任务与运动规划（TAMP）在确定性环境下表现良好，但在不确定性下容易因预测失败而频繁重新规划。
*   **研究动机**：人类在寻找物体时会利用常识（如：烤箱通常在厨房，相似的物体常放在一起）。手动为机器人编写这些复杂的空间关联规则极其困难。
*   **核心目标**：利用大语言模型（LLM）中蕴含的丰富常识，引导机器人的状态估计，从而在部分可观测的环境中实现更高效的规划与执行。

#### 2. 方法论：CoCo-TAMP 框架
CoCo-TAMP 结合了 LLM 的常识推理能力与严谨的贝叶斯滤波框架，主要包含以下关键技术：
*   **LLM 引导的初始信念生成**：通过多选题（MCQA）形式询问 LLM（如 GPT-4o），获取物体在特定房间或表面出现的先验概率，将 Logits 转换为初始信念分布。
*   **分层状态估计**：将信念分解为“房间-表面-连续位姿”三个层级。使用离散贝叶斯滤波处理语义位置，使用粒子滤波处理连续位姿。
*   **物体共存模型（Co-location Model）**：利用 LLM 生成物体的功能描述并提取文本嵌入（Embedding），计算物体间的余弦相似度。当观察到物体 A 时，根据相似度更新相似物体 B 的位置信念。
*   **共存触发器（Co-location Toggler）**：利用 LLM 判断某些物体是否具有“分散性”（如电灯开关通常分散在各处），从而决定是否禁用共存模型，避免错误的推理。
*   **可见性感知观测模型**：引入可见性项（Visibility），区分“没看到物体是因为它不在那”和“没看到是因为被遮挡了”，使信念更新更符合物理逻辑。
*   **规划器集成**：基于 PDDLStream，将观测动作的成本与信念质量挂钩，引导机器人优先探索高概率区域。

#### 6. 主要结论与发现
*   **效率提升**：CoCo-TAMP 在仿真中减少了 **62.7%** 的规划执行时间，在现实中减少了 **72.6%**。
*   **重规划次数**：显著降低了因观测失败导致的重规划次数，证明了 LLM 引导的信念更接近真实分布。
*   **LLM 的角色**：LLM 适合提供**先验（Priors）**，但在**信念更新（Update）**阶段，传统的贝叶斯滤波比纯 LLM 更新（LGBU）更鲁棒，尤其是在对抗性场景下。
*   **模型选择**：更强大的 LLM（如 GPT-4o）在提取常识方面明显优于较小的开源模型。

#### 7. 优点
*   **常识与逻辑的结合**：巧妙地将 LLM 的非结构化常识转化为结构化的概率分布，并嵌入到严谨的机器人规划框架中。
*   **通用性强**：不需要为每种环境手动建模物体关联，利用 LLM 的零样本（Zero-shot）能力即可处理海量物体类别。
*   **鲁棒性设计**：通过共存触发器和贝叶斯更新，解决了 LLM 可能产生的“幻觉”或错误先验问题。

#### 8. 不足与局限
*   **环境依赖**：假设已知环境的语义布局（房间和表面的位置），但在完全未知的环境中，机器人仍需先进行探索。
*   **动态环境**：目前主要针对静态物体，未考虑物体位置随时间动态变化的情况。
*   **实时性挑战**：依赖 LLM API 可能带来网络延迟和成本问题，且长程 TAMP 规划本身的计算复杂度依然较高。
*   **应用领域限制**：目前仅在家庭环境验证，在工厂、医院等专业领域的常识迁移能力尚待验证。
