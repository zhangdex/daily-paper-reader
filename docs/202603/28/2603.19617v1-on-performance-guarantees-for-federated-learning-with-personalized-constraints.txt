# On Performance Guarantees for Federated Learning with Personalized Constraints

Abstract 

Federated learning (FL) has emerged as a communication-efficient algorithmic framework for distributed learning across multiple agents. While standard FL formulations capture un-constrained or globally constrained problems, many practical settings involve heterogeneous resource or model constraints, leading to optimization problems with agent-specific feasible sets. Here, we study a personalized constrained federated optimization problem in which each agent is associated with a convex local objective and a private constraint set. We propose PC-FedAvg, a method in which each agent maintains cross-estimates of the other agentsŌĆÖ variables through a multi-block local decision vector. Each agent updates all blocks locally, penalizing infeasibility only in its own block. Moreover, the cross-estimate mechanism enables personalization without requiring consensus or sharing constraint information among agents. We establish communication-complexity rates of O(ŽĄŌłÆ2) for suboptimality and O(ŽĄŌłÆ1) for agent-wise infeasibility. Preliminary experiments on the MNIST and CIFAR-10 datasets validate our theoretical findings.

These considerations motivate the main contributions of this paper, which are fourfold: - We introduce a personalized constrained federated optimization formulation that captures heterogeneous agent-specific feasible sets while coupling agents through a regularized global objective. - We propose PC-FedAvg, in which each agent maintains cross-estimates of the other agentsŌĆÖ variables through a multi-block local decision vector. Each agent updates all blocks locally, penalizing infeasibility only in its own block. This design preserves constraint privacy, retains the standard server-agent communication structure, and enables personalization without enforcing consensus. - We derive lower and upper bounds of the global objective suboptimality and bounds on agent-wise infeasibility, with communication-complexity guarantees for both. In particular, we establish the best-known communication complexity [ 6 , 9] (O(ŽĄŌłÆ2)) for suboptimality, and 

O (ŽĄŌłÆ1) for infeasibility. - We validate the proposed method on MNIST and CIFAR-10 with heterogeneous Ōäō1 constraints and compare it with existing FL baselines. The literature on federated learning is vast; here, we focus on the lines of work most relevant to our setting. Constrained optimization has become an important direction in FL. Most existing approaches impose constraints on a shared global model, either through global constraints or identical per-agent constraints, and address them using projection-based, primalŌĆōdual, or penalty methods [ 10 , 11 ]. These methods preserve the classical FL structure but still require the overriding step mentioned earlier. Related projection-based methods for constrained distributed convex optimization with shared decision variables have been studied under various communication models, e.g., in [ 12 , 13 ], and more recently in [ 14 ], where a lazy step is used to mitigate projection-induced errors under constraints. In parallel, cross-estimate mechanisms have been explored in distributed Nash equilibrium seeking [ 15 ]. However, these approaches are not designed for FL and typically require frequent peer-to-peer communication, limiting their efficiency. Personalization in FL has been recently studied, particularly under statistical heterogeneity [ 8]. Existing personalized FL methods introduce agent-specific models through meta-learning and initialization strategies [ 16 ], regularization-based coupling [ 17 , 18 ], or shared representations with local components [ 19 , 20 ]. While effective for handling data heterogeneity, these approaches do not explicitly enforce agent-specific feasibility constraints.

2.2 Algorithm Outline 

The PC-FedAvg algorithm is summarized in Algorithm 1. Each agent maintains a multi-block decision vector, with one block per agent. During local updates, every agent updates all blocks. Remark 3.5. Theorem 3.4 provides communication-complexity guarantees for solving problem (1) .This setting is fundamentally different from classical constrained FL, which typically imposes a single shared constraint on a global model. Despite this additional heterogeneity and the absence of consensus, the theorem shows that the proposed block-wise algorithm with cross-estimates achieves the same order of communication complexity for optimality as standard unconstrained stochastic FL, namely O(ŽĄŌłÆ2) for suboptimality, while simultaneously driving each agent to feasibility with a rate 

O(ŽĄŌłÆ1). In particular, Theorem 3.4 implies that the presence of personalized constraints does not degrade the order-wise communication complexity compared with classical federated averaging-type methods. 

Remark 3.6. Since problem (1) is constrained, its optimal solution xŌłŚ does not necessarily minimize the unconstrained objective f . In particular, f (xŌłŚ) need not attain the global minimum of f , so we also require an appropriate lower bound for the optimality metric. The infeasibility bound exhibits the standard penalty-parameter tradeoff: a larger Žü enforces feasibility more strongly, but it can worsen the suboptimality bound. We derive the choice Žü := ŌłÜR that balances these competing effects. In particular, with this choice we retain the standard O(ŽĄŌłÆ2) communication complexity for suboptimality, while obtaining a better O(ŽĄŌłÆ1) communication complexity for agent-wise infeasibility relative to suboptimality.

Although we solve a constrained problem, our method achieves global objective values that are comparable to the baselines, especially on MNIST (Fig. 3). In terms of infeasibility, however, our method consistently outperforms the baselines. As shown in Figs. 1 and 2, even when all methods are initialized from a feasible point, the baseline methods eventually drift toward infeasible iterates, whereas our iterates remain feasible throughout. We emphasize that, even when a penalty on the distance to each agentŌĆÖs feasible set is added to the objective, the baseline methods remain centered on learning a single shared model. This is because, at the beginning of each communication round, all agents are initialized from the same global model: the server overwrites local models with their average. In contrast, the block-wise structure of our method allows different blocks to remain distinct across agents over rounds. Specifically, the server updates only the blocks corresponding to each agent rather than fully averaging all blocks. As a result, agents start each round from different model states, which is desirable for personalization. ## 6 CONCLUSION 

We studied a personalized federated optimization problem in which each agent maintains a locally constrained decision variable, while cooperation is achieved through an objective evaluated at the population average. We proposed a projection-based multi-block federated algorithm that enforces local feasibility at every iteration while preserving privacy and the standard serverŌĆōagent communication pattern. We established convergence guarantees for both suboptimality and feasibility that match the overall convergence rate of unconstrained FL methods. Experiments on MNIST and CIFAR-10 with heterogeneous Ōäō1 constraints support the theory and demonstrate competitive performance against standard federated baselines.
