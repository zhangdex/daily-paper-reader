## 摘要
我们提出了一种新的语言强化学习（Verbal Reinforcement Learning, VRL）框架，用于在执行不确定性下运行的移动机器人系统中进行可解释的任务级规划。该框架采用闭环架构，通过与物理环境的交互实现迭代策略改进。在我们的框架中，可执行的行为树（Behavior Trees）由大语言模型（LLM）执行器（actor）进行反复优化，该执行器使用由视觉语言模型（VLM）评论器（critic）生成的结构化自然语言反馈，而评论器负责观察物理机器人和执行轨迹。与传统的强化学习不同，VRL 中的策略更新直接发生在符号规划层面，无需基于梯度的优化。这实现了透明的推理、显式的因果反馈以及人类可解释的策略演化。我们在执行不确定性下执行多阶段操作和导航任务的真实移动机器人上验证了所提出的框架。实验结果表明，该框架支持可解释的策略改进、对执行失败的闭环自适应以及在物理机器人系统上的可靠部署。

## Abstract
We propose a new Verbal Reinforcement Learning (VRL) framework for interpretable task-level planning in mobile robotic systems operating under execution uncertainty. The framework follows a closed-loop architecture that enables iterative policy improvement through interaction with the physical environment. In our framework, executable Behavior Trees are repeatedly refined by a Large Language Model actor using structured natural-language feedback produced by a Vision-Language Model critic that observes the physical robot and execution traces. Unlike conventional reinforcement learning, policy updates in VRL occur directly at the symbolic planning level, without gradient-based optimization. This enables transparent reasoning, explicit causal feedback, and human-interpretable policy evolution. We validate the proposed framework on a real mobile robot performing a multi-stage manipulation and navigation task under execution uncertainty. Experimental results show that the framework supports explainable policy improvements, closed-loop adaptation to execution failures, and reliable deployment on physical robotic systems.

#### 2. 方法论
该论文提出了一种**言语强化学习（Verbal Reinforcement Learning, VRL）**框架，其核心思想是将策略表示为符号化的**行为树（Behavior Trees, BTs）**，并通过自然语言反馈进行迭代优化。

*   **LLM 执行器（Actor）**：负责修改行为树。它接收任务定义、环境规范、节点库、当前行为树、评论器的语言反馈以及数值评分，通过离散的符号编辑（而非梯度更新）来优化策略。
*   **VLM 评论器（Critic）**：利用视觉语言模型观察执行过程。它分析机器人操作的视觉快照和行为树执行轨迹，输出结构化的自然语言反馈、警报分数（$s$）和置信度（$c$）。
*   **算法流程**：
    1.  执行当前行为树。
    2.  VLM 评论器在初始、中间和最终阶段进行观察。
    3.  如果检测到严重问题（警报分数高），通知人工干预。
    4.  收集最终反馈和真实得分。
    5.  LLM 执行器根据反馈更新行为树，存入记忆库并进入下一轮迭代。

#### 6. 主要结论与发现
*   **反馈质量胜过模型规模**：经过领域微调的 3B 模型在策略改进的稳定性上优于未经微调的 7B 模型，说明评论器与任务领域的对齐比参数量更重要。
*   **因果诊断是关键**：单纯的数值奖励无法区分硬件错误和规划错误，结构化的语言反馈对于稳定改进至关重要。
*   **感知是瓶颈**：引入符号化的感知信息（如块颜色）能显著加速收敛，说明 VRL 的主要限制在于视觉感知的准确性，而非符号规划能力。
*   **自适应行为**：执行器会根据评论器的局限性调整策略（例如，当视觉不可靠时，执行器会选择更保守的动作序列）。

#### 7. 优点
*   **高度可解释性**：策略更新是人类可读的行为树修改，而非黑盒参数调整。
*   **无需梯度**：在符号层面进行优化，避免了传统 RL 极其耗时的训练过程和对精确模拟器的依赖。
*   **闭环自适应**：通过 VLM 实现了从物理世界到逻辑规划的反馈闭环，能有效处理现实中的执行失败。

#### 8. 不足与局限
*   **感知依赖性**：系统性能高度依赖 VLM 的视觉接地（Grounding）能力，若评论器看错，可能导致错误的策略修改。
*   **时间窗口限制**：目前的评论器主要基于快照，难以处理具有长程依赖关系的潜在故障（即错误在很久之后才显现）。
*   **策略保守性**：在感知不确定时，Actor 倾向于采取保守策略（如禁用某些功能），这可能导致系统陷入次优解。
*   **实时性**：调用大型多模态模型作为评论器和执行器可能存在推理延迟，影响实时响应能力。
