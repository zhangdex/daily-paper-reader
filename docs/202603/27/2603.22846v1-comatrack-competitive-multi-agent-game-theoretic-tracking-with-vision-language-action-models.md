## 摘要
具身视觉追踪（EVT）是具身智能中的核心动态任务，要求智能体精确跟随语言指定的目标。然而，大多数现有方法依赖于单智能体模仿学习，面临专家数据成本高昂以及由于静态训练环境导致的泛化能力有限的问题。受竞争驱动的能力演化启发，我们提出了 CoMaTrack，这是一个竞争性博弈论多智能体强化学习框架，在具有竞争性子任务的动态对抗设置中训练智能体，从而产生更强的自适应规划和抗干扰策略。我们进一步推出了 CoMaTrack-Bench，这是首个竞争性 EVT 基准测试，其特点是追踪器与自适应对手在不同环境和指令下的博弈场景，实现了主动对抗交互下的标准化鲁棒性评估。实验表明，CoMaTrack 在标准基准测试和 CoMaTrack-Bench 上均达到了最先进的结果。值得注意的是，使用我们框架训练的 3B VLM 在具有挑战性的 EVT-Bench 上超过了之前基于 7B 模型的单智能体模仿学习方法，在 STT、DT 和 AT 指标上分别达到了 92.1%、74.2% 和 57.5%。基准测试代码将在 https://github.com/wlqcode/CoMaTrack-Bench 提供。

## Abstract
Embodied Visual Tracking (EVT), a core dynamic task in embodied intelligence, requires an agent to precisely follow a language-specified target. Yet most existing methods rely on single-agent imitation learning, suffering from costly expert data and limited generalization due to static training environments. Inspired by competition-driven capability evolution, we propose CoMaTrack, a competitive game-theoretic multi-agent reinforcement learning framework that trains agents in a dynamic adversarial setting with competitive subtasks, yielding stronger adaptive planning and interference-resilient strategies. We further introduce CoMaTrack-Bench, the first benchmark for competitive EVT, featuring game scenarios between a tracker and adaptive opponents across diverse environments and instructions, enabling standardized robustness evaluation under active adversarial interactions. Experiments show that CoMaTrack achieves state-of-the-art results on both standard benchmarks and CoMaTrack-Bench. Notably, a 3B VLM trained with our framework surpasses previous single-agent imitation learning methods based on 7B models on the challenging EVT-Bench, achieving 92.1% in STT, 74.2% in DT, and 57.5% in AT. The benchmark code will be available at https://github.com/wlqcode/CoMaTrack-Bench

*   **现有挑战**：主流方法依赖**单智能体模仿学习（IL）**，存在专家数据获取成本高、在静态环境中训练导致泛化性差、难以应对遮挡或干扰等问题。
*   **研究动机**：受生物竞争进化启发，作者认为单智能体强化学习（RL）若缺乏不断增强的外部压力，容易陷入局部最优。因此，论文提出通过**竞争性博弈**让智能体在对抗中进化，从而获得更强的预测、重定位和抗干扰能力。

#### 2. 方法论
CoMaTrack 采用两阶段训练框架，将 EVT 建模为多智能体对抗博弈：
*   **核心架构**：基于 **Qwen2.5VL-3B** 多模态大模型，集成 **Flow-matching** 动作头，直接从多视角图像和历史序列预测未来 5 步的连续路标（Waypoints）。
*   **第一阶段：监督微调（SFT）**：利用导航、追踪和 VQA 多任务数据进行对齐，使模型具备基础的指令理解和空间感知能力。
*   **第二阶段：多智能体强化学习（MARL）**：
    *   **算法**：采用 **GRPO（组相对策略优化）** 算法。
    *   **对抗机制**：同时训练“追踪者（Tracker）”和“对手（Opponent）”。对手的目标是抢占目标附近的有利位置，从而通过物理遮挡、路径交叉等方式干扰追踪者。
    *   **非对称奖励设计**：追踪者奖励侧重于保持安全距离（1-3m）和避障；对手奖励则鼓励其更靠近目标（约 1.25m），迫使追踪者学习更复杂的避让和重新锁定策略。

*   **数据集/场景**：使用 Habitat 3.0 模拟器，场景源自 HM3D 和 MP3D。
*   **基准测试（Benchmark）**：
    1.  **EVT-Bench**：包含单目标追踪（STT）、干扰追踪（DT）和歧义追踪（AT）。

*   **对比方法**：对比了 TrackVLA、TrackVLA++、Uni-NaVid、VLingNav 等 SOTA 方法，其中许多基准方法使用 7B 参数模型。

#### 6. 主要结论与发现
*   **竞争驱动进化**：多智能体对抗训练显著提升了追踪成功率（SR）并降低了碰撞率（CR）。

*   **鲁棒性增强**：在面对主动干扰（如对手故意遮挡）时，CoMaTrack 表现出更强的预测能力和路径规划灵活性。

#### 7. 优点
*   **范式创新**：首次将多智能体博弈论引入具身视觉追踪，解决了静态数据训练的局限性。
